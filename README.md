# :clap: Clappity

Clappity makes the lives of people easier through the use of an IoT device. It is an easy way to order online, anytime from your couch. 

Want a pizza? It's just a clap away.

## Features
- 2x clap for pizza

## Tech Stack
- Node.js
- Tessel (Micro-controller)
- WebSockets
- Twilio
